# open-japanese-mesh

## setup
`$ pipenv install`
`$ pipenv shell`


## how to use
`$ pipenv run jupyter notebook`
- see mannual.ipynb


## objective
- try to map Japanese biomedical terms to MeSH IDs via open Japanese-English glossaries
- 様々な日英辞書とMeSHを結びつける
