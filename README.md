# open-japanese-mesh

## objectives
- map Japanese biomedical terms to the IDs of various ontologies
- submit to PubDictionary


## setup
```
$ pipenv install
$ pipenv shell
```


## how to use
```
$ pipenv run jupyter notebook
```
see mannual.ipynb
